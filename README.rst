A library to simulate a closed-loop system. Includes :

* Kalman filter
* Mahony and Madgwick estimators
* PID controller
* FIR filter
* system simulation
* sensor simulation (with bias and noise)


Free software: MIT license

Documentation: https://systemcontrol.readthedocs.io.
